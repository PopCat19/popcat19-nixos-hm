# Features Overview

This document provides a comprehensive overview of all features included in this NixOS configuration.

## Desktop Environment

### Hyprland Wayland Compositor
- **Modern Wayland compositor** with advanced features
- **Rose Pine theming** consistent across all applications
- **Custom keybindings** optimized for productivity
- **Window rules** for specific applications
- **Multi-monitor support** with per-display configuration
- **GPU acceleration** for smooth performance

### Display Manager
- **SDDM** with Rose Pine theming
- **Automatic login** configuration
- **Session management** for Hyprland

### Application Launcher
- **Fuzzel** - Fast, lightweight application launcher
- **Rose Pine themed** interface
- **Custom keybindings** for quick access

### Status Bar
- **QuickShell** - Modern status bar
- **System monitoring** (CPU, memory, network)
- **Media controls** and notifications
- **Custom widgets** for various functions
